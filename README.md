# dotenv-types

dotenv-types is a CLI for generating .env typings for Typescript

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dotenv-types.

```bash
$ npm i -g dotenv-types
```

## Usage

```bash
$ dotenv-types -h
```
